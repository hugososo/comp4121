## Installation and Run

### Client
You need to have NodeJs installed first

```
cd client
npm i
npm install react-router-dom
npm run dev
```

### Server
You need to have NodeJs installed first

You should have OpenAI API KEY first
```
cd server
touch .env
```
Put you API inside
```
cd server
npm i
npm run start
```
